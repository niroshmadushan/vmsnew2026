## 🔧 How It Works

### Signup Flow:

1. **User fills signup form** → `app/auth/secure-signup/page.tsx`
2. **Frontend calls backend API** → `https://saasapi.cbiz365.com/api/auth/secure-signup`
   - Role is set to `'staff'` (hardcoded)
   - Backend API receives signup request
3. **Backend generates verification email**:
   - Backend API creates user account
   - Backend API generates verification link
   - Verification link should use: `https://saasapi.cbiz365.com/verify-email`
4. **User receives email** with verification link

### Role Configuration:

**Before:**


**After:**
