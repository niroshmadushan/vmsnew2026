📬 Email Recipients

The email is sent to the following participants:

  • ✅ Responsible Person - The person who created/manages the booking
  • ✅ Internal Members - Staff/employees added as participants
  • ✅ External Members - External visitors/guests added as participants

Note: Only participants with valid email addresses receive the email.

📋 Email Content Fields

The email includes the following information:

  • 📅 Booking Title/Meeting Name
  • 🔖 Booking Reference ID (if available)
  • 📅 Date (formatted as: "Monday, January 15, 2024")
  • 🕐 Time (formatted as: "10:00 - 12:00")
  • 📍 Location/Place
  • 🏢 Address (if available)
  • 📞 Phone (if available)
  • 📝 Description (if available)
  • 💬 Additional Message (if custom message is provided)
  • 📅 Calendar Invitation (ICS file attachment - only in frontend template)
